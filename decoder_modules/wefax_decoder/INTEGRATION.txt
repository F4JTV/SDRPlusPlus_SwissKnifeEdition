Integrating the WEFAX module into the SDR++ build
==================================================

1. Copy the `wefax_decoder/` folder into `SDRPlusPlus/decoder_modules/`.

2. Edit the top-level `SDRPlusPlus/CMakeLists.txt` and add the option (near the
   other OPT_BUILD_*_DECODER options):

       option(OPT_BUILD_WEFAX_DECODER "Build the WEFAX / HF radiofax decoder module (no dependencies required)" ON)

3. Further down, near the other decoder_modules add_subdirectory() blocks, add:

       if (OPT_BUILD_WEFAX_DECODER)
       add_subdirectory("decoder_modules/wefax_decoder")
       endif (OPT_BUILD_WEFAX_DECODER)

4. Build:

       cd SDRPlusPlus/build
       cmake ..
       make wefax_decoder -j$(nproc)

   The resulting plugin is:
       build/decoder_modules/wefax_decoder/wefax_decoder.so

5. Install it (or use the prebuilt one under prebuilt-ubuntu24/):

       sudo cp build/decoder_modules/wefax_decoder/wefax_decoder.so /usr/lib/sdrpp/plugins/

   Then enable "wefax_decoder" in the SDR++ module manager.
